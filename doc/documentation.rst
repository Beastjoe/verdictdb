Connecting to Databases using Veeline
======================================

Veeline is a command-line interface that ships with Verdict. You can connect to
your databases (e.g., Hive, Impala, MySQL) using veeline and issue analytic
queries.

    :maxdepth: 2

    documentation/veeline_to_databases.rst
