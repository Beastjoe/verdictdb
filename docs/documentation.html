Connecting to Databases using Veeline¶

Veeline is a command-line interface that ships with Verdict. You can connect to your databases (e.g., Hive, Impala, MySQL) using veeline and issue analytic queries.

  • Connecting to Databases using Veeline

Connecting to Databases using Verdict’s JDBC Driver¶

Verdict provides the standard JDBC protocol. The existing application that makes JDBC connections to databases requires minimal changes.
